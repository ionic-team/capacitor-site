# Capacitor CLI - cap

View all available CLI commands and options.

```bash
npx cap [-V] [-h]
```

<strong>Options:</strong>

- `-V, --version` (optional): Output the version number
- `-h, --help` (optional): Output usage information. Can be used with individual commands too.
